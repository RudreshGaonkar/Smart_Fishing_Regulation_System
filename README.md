# Smart Fishing Regulation System 🐟⚓
A web app that protects marine life, guides fishermen safely, and streamlines coastal management.


**The Problem**

Overfishing is wiping out endangered species. No real-time monitoring means chaos at sea.


**The Solution**

Connects Fishermen, Researchers, and Admins on one platform:
- Fishermen log trips + get smart routes avoiding no-go zones
- Admins track everything live on an interactive map
- Alerts prioritize high-risk areas


**Tech Stack**

Frontend: React + TypeScript

Backend: Node.js + MySQL

Key Algorithms:
- Graph + BFS for safe routing
- Priority Queue for risk alerts
